Semana 2 y 3

Operaciones Vectoriales y Unitarios

Fundamentos para la suma de fuerzas y representación de dirección.

A · B = AxBx + AyBy + AzBz
λ = F / ||F|| = cos θx i + cos θy j + cos θz k

Variables: λ: Vector Unitario ||F||: Magnitud Ax, Ay, Az: Componentes

Semana 4

Fuerzas en el Espacio (3D)

Uso de Cosenos Directores para la resolución de componentes tridimensionales.

Fx = F cos θx | Fy = F cos θy | Fz = F cos θz
cos² θx + cos² θy + cos² θz = 1

Aplicación: Determinar la orientación de una carga en estructuras espaciales mediante los ángulos directores.

Semana 4

Equilibrio de la Partícula

Aplicación de la Primera Ley de Newton para sistemas en reposo.

ΣF = 0   ⮕   ΣFx = 0, ΣFy = 0, ΣFz = 0

Metodología DCL:

  1. Aislar la partícula de su entorno.
  2. Identificar todas las fuerzas (Peso, Normal, Tensión, Fricción).
  3. Establecer ejes de referencia (x, y, z).
  4. Resolver sistema de ecuaciones lineales.
