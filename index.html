1. Herramientas Vectoriales

Fundamentos matemáticos para la descomposición y análisis de fuerzas en elementos estructurales.

Producto Escalar (Punto)

A · B = |A| |B| cos(θ)

Aplicación: Útil para hallar la proyección de una fuerza sobre una barra o el ángulo entre dos cables.

A, B: Magnitudes θ: Ángulo Ax, Ay: Componentes

Producto Vectorial (Cruz)

M = r × F

Aplicación: Cálculo de momentos de fuerza respecto a un punto, clave para analizar la tendencia al giro de un sólido.

2. Equilibrio del Cúerpo Sólido

ΣF = 0   |   ΣM = 0

Definición: Condición fundamental donde la resultante de fuerzas y momentos es nula, asegurando la estabilidad del sistema.

En el plano (2D): ΣFx = 0 ΣFy = 0.

3. Metodología: DCL

Procedimiento sistemático para resolver problemas de equilibrio:

  1. Aislar: Representar el nodo o sólido como un punto.
  2. Diagramar: Dibujar todas las fuerzas externas y reacciones.
  3. Ejes: Definir coordenadas alineadas a los apoyos.
  4. Resolver: Aplicar ΣF=0 para despejar incógnitas.

4. Tipos de Cargas Comunes

  • Peso (W = m·g): Carga gravitacional vertical.
  • Normal (N): Reacción perpendicular a la superficie.
  • Tensión (T): Fuerza axial dirigida a lo largo del cable.
  • Fricción (f): Fuerza tangencial opuesta al deslizamiento.
