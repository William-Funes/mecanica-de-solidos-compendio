1. Análisis Vectorial Completo

Producto Escalar (Punto) 2D / 3D

Aplicación: Cálculo del ángulo entre vectores y proyecciones de fuerza.

A · B = AxBx + AyBy + AzBz
A · B = |A| |B| cos(θ)

A, B = Magnitudes θ = Ángulo Ax, Ay, Az = Componentes

Producto Vectorial (Cruz)

Aplicación: Determinación de momentos de fuerza (torque).

a × b = (aybz - azby)i - (axbz - azbx)j + (axby - aybx)k
|a × b| = |a| |b| sen(α)

2. Resultantes de Sistemas de Fuerzas

Método Analítico (Componentes)

Aplicación: Obtención de la fuerza única equivalente.

Rx = ΣFix = Σ(Fi cos θi)
Ry = ΣFiy = Σ(Fi sen θi)
R = √(Rx² + Ry²) | θR = arctan(Ry / Rx)

Magnitud en el Espacio (3D)

||F|| = √(Fx² + Fy² + Fz²)

Aplicación: Modulo del vector fuerza en un sistema tridimensional.

Método Trigonométrico Ley del Coseno

R = √(F1² + F2² + 2F1F2 cos θ)

Aplicación: Resultante de dos fuerzas concurrentes con ángulo θ.

tan α = (F2 sen θ) / (F1 + F2 cos θ)

3. Equilibrio de la Partícula

La condición de equilibrio exige que la resultante sea igual a cero.

ΣFx = 0 | ΣFy = 0 | ΣFz = 0
R = ΣFi = 0

4. Caracterización y Metodología

Tipos de Fuerzas:

  • Peso: W = m · g (Fuerza vertical hacia abajo)
  • Normal (N): Reacción perpendicular a la superficie
  • Tensión (T): Transmitida por cables (siempre a lo largo del cable)
  • Fricción (f): Paralela a la superficie, se opone al movimiento

Procedimiento DCL:

  1. Aislar la partícula (punto).
  2. Dibujar el DCL con todas las fuerzas, magnitudes y ángulos.
  3. Elegir sistema de coordenadas (x, y, z).
  4. Descomponer fuerzas y aplicar ecuaciones de equilibrio.
