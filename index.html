I. Análisis Vectorial de Cargas

Producto Escalar (Punto)

A · B = AxBx + AyBy + AzBz = |A||B| cos(θ)

Aplicación: Determinación del ángulo entre fuerzas o proyección de una carga sobre un eje estructural[cite: 5, 8].

Variables: A, B (magnitudes), θ (ángulo), Ax, Ay, Az (componentes)[cite: 4, 5].

Producto Vectorial (Cruz)

a × b = det | i j k ; ax ay az ; bx by bz |

Aplicación: Cálculo de momentos (torque) en soportes y elementos sólidos[cite: 10].

Variables: i, j, k (vectores unitarios), a, b (vectores de posición o fuerza)[cite: 10].

II. Equilibrio del Cuerpo Sólido

Condición de Equilibrio

ΣFx = 0 | ΣFy = 0 | ΣFz = 0 | ΣM = 0

Aplicación: Garantizar la estabilidad de la partícula o sólido antes del análisis de esfuerzos internos[cite: 31, 38, 70].

Cálculo de la Resultante

R = √(Rx² + Ry²) | θ = arctan(Ry/Rx)

Aplicación: Simplificación de múltiples fuerzas concurrentes en una carga única[cite: 47, 49].

III. Caracterización de Fuerzas

  • Peso (W = m·g): Fuerza vertical por gravedad[cite: 73, 74].
  • Tensión (T): Fuerza axial en cables o cuerdas[cite: 76, 77].
  • Normal (N): Reacción perpendicular a la superficie[cite: 75].
  • Fricción (f): Fuerza tangencial opuesta al movimiento[cite: 79, 81].

IV. Metodología de Resolución (DCL)

  1. Aislar el elemento sólido o partícula[cite: 86].
  2. Dibujar el Diagrama de Cuerpo Libre (DCL) con todas las fuerzas y ángulos[cite: 89, 90].
  3. Establecer el sistema de coordenadas (x, y, z)[cite: 92, 93].
  4. Descomponer fuerzas inclinadas y aplicar ΣF = 0[cite: 95, 99].
